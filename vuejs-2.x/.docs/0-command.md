# vuejs-2.x

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```

### Lints and fixes files

```
npm run lint
```

### alias 생성

- jsconfig.json 파일 생성
- build/serve 명령어 실행시 자동으로 실행됨

```
npm run aliases
```

### static-server

- 빌드 후 서버상에서 실행하여 보고 싶은 경우
- 데이터 연동은 안됨

```
npm run static
```
